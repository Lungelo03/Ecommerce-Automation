# AutomationExercise Test Automation Project

This repository contains an automated test suite for **[AutomationExercise](https://automationexercise.com)**, a demo e-commerce website widely used for practicing UI and API test automation.

The project is designed to validate core user flows such as authentication, product browsing, cart operations, and checkout, following industry-standard automation practices.
